This is my first job interview task to do 3d viewer open book assessment, and i done that immediately and my manager suprised and give me a intern role.

This repo is created by following this documenttion  https://tutorials.autodesk.io/tutorials/simple-viewer/  and https://tutorials.autodesk.io/tutorials/dashboard/
To run this clone this project
create .env file in the root folder and insert your client_id and client_secret
Then in your terminal write the command "npm install".

In your browser goto  http://localhost:3001
